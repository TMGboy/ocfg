## API Reference

#### Get all categories

```bash
  GET /categories
```

#### Get category channels

```bash
  GET /categories/${id}/channels
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of category  |

#### Get channel

```bash
  GET /channel/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of channel  |
